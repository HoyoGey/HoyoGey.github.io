Документация по использованию калькулятора производных

Основные правила ввода

  • Используйте латинские буквы x и y для переменных
  • Для возведения в степень используйте символ ^ (например: x^2)
  • Для умножения используйте символ * (например: 2*x)
  • Для деления используйте символ / (например: x/y)

Поддерживаемые правила дифференцирования

1. Правило суммы/разности

(a±u)' = u±u'

Пример: (x^7 + 3x^5)' = 7x^6 + 15x^4

2. Правило произведения

(u*v)' = u'v + uv'

Пример: (3x^4 * y^6)' = (3x^4)' * y^6 + 3x^4 * (y^6)' = 12x^3 * y^6 + 3x^4 * 6y^5

3. Правило частного

(u/v)' = (u'v - uv')/v^2

Пример: (3x^4/y^6)' = ((3x^4)' * y^6 - 3x^4 * (y^6)')/(y^6)^2 = (12x^3 * y^6 - 3x^4 * 6y^5)/(y^6)^2

Примеры использования

1. Простая степенная функция:

x^4

2. Произведение функций:

3x^4 * y^6

3. Частное функций:

3x^4/y^6
Примечание: Калькулятор показывает полное пошаговое решение, включая применение правил дифференцирования и промежуточные вычисления.
